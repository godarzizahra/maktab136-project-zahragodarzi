# Watch Store

An e-commerce watch store built with Next.js.

## Features

- Product listing
- Product details page
- User authentication
- Shopping cart
- Wishlist
- Responsive design
- Dark mode

## Technologies

- Next.js
- React
- Zustand
- Axios
- Tailwind CSS
